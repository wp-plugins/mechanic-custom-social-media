== Description ==

Mechanic Custom Social Media is a widgets which will display your social media page (External Links) on WordPress. When you hover a icon, a small tooltip is displayed with the name of the social media. All other icons have a low opacity.

== Installation ==

This section describes how to install the plugin and get it working.

1. Download Simple Custom Social Media WordPress Plugins
2. Upload `mechanic-socialmedia.zip` to the `/wp-content/plugins/` directory. You can do this using 'Upload' functionality provided in plugins section of your wordpress dashboard
3. Activate the plugin through the `Plugins` menu in WordPress
4. Go to `Appearance` >> `Widgets` and drag `Mechanic - Custom Social Media` in to your widget area.
5. Save
7. Go to Appreance >> Widgets >> drag "Mechanic - Custom Social Media" to your widgets area position.
8. Fill all available text field correctly and all will be an "external link" with nofollow rel.
9. You are done.

= How to editing replace image? =

you must open the file `wp-mechanicsocialmedia.php` plugins on the `wp-content/plugins/mechanic-socialmedia/` and find the css listing after SOCIAL MEDIA ICON - CSS and replace the following external link image with your image URL.
